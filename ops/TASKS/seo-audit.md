# Task Playbook: SEO Audit (technical + copy)

## Output (publish-ready)
- Executive summary (what to do first, why, expected impact)
- Technical findings (crawl/indexing, CWV/perf, schema, canonicals, sitemaps)
- Content findings (pages to update, keyword intent gaps, internal links)
- Prioritized backlog (P0/P1/P2)
- Evidence links/screenshots/export references

## Required artifacts per run
- `00_intake.md`
- `01_context.md` (site, GSC properties, crawl scope)
- `02_plan.md` (acceptance criteria)
- `04_worklog.md`
- `05_outputs/seo-audit.md`
- `06_qa_report.md`

## QA checklist
- [ ] No invented metrics (cite source/export)
- [ ] Recommendations are actionable (who/what/where)
- [ ] Priorities justified
- [ ] Copy suggestions match brand voice constraints
