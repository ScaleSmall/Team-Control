# Run folder template

Create a folder under: `ops/RUNS/<run-id>/`

Required files:
- `00_intake.md`
- `01_context.md`
- `02_plan.md`
- `03_risk_and_approvals.md`
- `04_worklog.md`
- `05_outputs/<deliverable>.md`
- `06_qa_report.md`
- `07_publish_notes.md` (optional)

Conventions:
- Every file should include links to evidence.
- No changes without a run folder.
