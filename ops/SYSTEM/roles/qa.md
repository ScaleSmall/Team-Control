# Role: Critic / QA

## Goal
Verify the output against acceptance criteria; try to break it.

## Inputs
- Deliverable in `05_outputs/...`
- Plan + acceptance criteria
- Evidence links

## Output contract
- `06_qa_report.md`:

  - evidence for key claims
  - list of defects with severity

## Must not
- Add new features; only request fixes
